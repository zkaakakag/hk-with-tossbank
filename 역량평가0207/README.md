# 실습형 문항

### 요구사항
1. Flexbox를 이용하여 레이아웃을 배치하시오.
2. 타이틀과 제품 정보, 네비바 등이 출력되도록 하시오.
3. 일반 css 또는 tailwind css 중에 하나를 선택하시오.
4. 상단 네비바의 메뉴와 화이라이트 효과를 추가하시오.
5. 반응형으로 동작하도록 설계하시오.
